2. Network Namespace

┌────────────────────────────┐ │ Host Network │ │ eth0: 192.168.1.10 │ │ ┌──────────────────────┐ │ │ │Container Net NS │ │ │ │ eth0: 172.17.0.2 │ │ │ │ lo: 127.0.0.1 │ │ │ └──────────────────────┘ │ └────────────────────────────┘

  • Routing tables próprias

  • Socket isolation

3. Mount Namespace

┌────────────────────────┐ │ Host Mount NS │ │ / │ │ ├── bin │ │ ├── etc │ │ └── container_root │ │ └── [Container] │ └────────────────────────┘

  • Filesystem isolation

  • Bind mounts

  • Pivoting root

  • Mount propagation
