O que é uma Imagem Docker?

Uma imagem Docker é um pacote leve, autônomo e executável que inclui tudo necessário para rodar uma aplicação:

  • Código

  • Runtime

  • Bibliotecas

  • Variáveis de ambiente

  • Arquivos de configuração
