Pré-História dos Containers 📜

1979: Unix V7

  • Introdução do chroot

  • Primeiro passo para isolamento de processos

  • Limitava visão do sistema de arquivos

Anos 90: Primeiros Passos

  • FreeBSD Jails

  • Isolamento mais robusto

  • Separação de recursos
